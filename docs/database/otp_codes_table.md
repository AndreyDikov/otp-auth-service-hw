# 📋 otp_codes

> `otp_codes` — таблица для хранения OTP-кодов, привязанных к пользователю и операции. Код хранится в виде хэша, а 
> статус записи отражает текущее состояние OTP-кода: активен, использован или просрочен

## 🧾 Поля таблицы

| Поле           | Тип            | Может быть `null` | Значение по умолчанию          | Описание                                       |
|----------------|----------------|------------------:|--------------------------------|------------------------------------------------|
| `id`           | `bigint`       |                 ❌ | `generated always as identity` | Уникальный идентификатор OTP-кода              |
| `user_id`      | `bigint`       |                 ❌ | —                              | Идентификатор пользователя                     |
| `operation_id` | `varchar(255)` |                 ❌ | —                              | Идентификатор операции, к которой привязан код |
| `code_hash`    | `varchar(255)` |                 ❌ | —                              | Хэш OTP-кода                                   |
| `status`       | `varchar(20)`  |                 ❌ | —                              | Текущий статус OTP-кода                        |
| `expires_at`   | `timestamp`    |                 ❌ | —                              | Дата и время истечения срока действия кода     |
| `created_at`   | `timestamp`    |                 ❌ | `now()`                        | Дата и время создания записи                   |
| `used_at`      | `timestamp`    |                 ✅ | —                              | Дата и время успешного использования кода      |

---

## 🔒 Ограничения таблицы

| Ограничение            | Поле / поля | Описание                                                                                         |
|------------------------|-------------|--------------------------------------------------------------------------------------------------|
| `pk_otp_codes`         | `id`        | Первичный ключ таблицы. Значение `id` уникально для каждой записи                                |
| `fk_otp_codes_users`   | `user_id`   | Внешний ключ на таблицу `users`. При удалении пользователя связанные OTP-коды удаляются каскадно |
| `chk_otp_codes_status` | `status`    | Разрешены только значения: `ACTIVE`, `EXPIRED`, `USED`                                           |
