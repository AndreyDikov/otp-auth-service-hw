# 📋 notification_outbox

> `notification_outbox` — таблица для хранения задач на отправку OTP-кодов через внешние каналы. Записи сначала
> создаются в статусе `PENDING`, затем scheduler забирает их в обработку, отправляет код и переводит в `SENT`, либо
> возвращает в очередь/помечает как `FAILED` при ошибках

## 🧾 Поля таблицы

| Поле                   | Тип            | Может быть `null` | Значение по умолчанию          | Описание                                                                            |
|------------------------|----------------|------------------:|--------------------------------|-------------------------------------------------------------------------------------|
| `id`                   | `bigint`       |                 ❌ | `generated always as identity` | Уникальный идентификатор записи outbox                                              |
| `notification_channel` | `varchar(20)`  |                 ❌ | —                              | Канал отправки OTP-кода                                                             |
| `destination`          | `varchar(255)` |                 ❌ | —                              | Получатель уведомления                                                              |
| `encrypted_code`       | `text`         |                 ❌ | —                              | Зашифрованный OTP-код, который будет расшифрован перед отправкой                    |
| `status`               | `varchar(20)`  |                 ❌ | `'PENDING'`                    | Текущий статус обработки записи                                                     |
| `attempts`             | `int`          |                 ❌ | `0`                            | Количество неуспешных попыток отправки                                              |
| `error_message`        | `text`         |                 ✅ | —                              | Текст последней ошибки при отправке уведомления                                     |
| `created_at`           | `timestamp`    |                 ❌ | `now()`                        | Дата и время создания записи                                                        |
| `processed_at`         | `timestamp`    |                 ✅ | —                              | Дата и время финальной обработки записи: успешной отправки или окончательной ошибки |
|

---

## 🔒 Ограничения таблицы

| Ограничение                        | Поле / поля            | Описание                                                             |
|------------------------------------|------------------------|----------------------------------------------------------------------|
| `pk_notification_outbox`           | `id`                   | Первичный ключ таблицы. Значение `id` уникально для каждой записи    |
| `chk_notification_outbox_channel`  | `notification_channel` | Разрешены только значения: `EMAIL`, `SMS`, `TELEGRAM`, `FILE`        |
| `chk_notification_outbox_status`   | `status`               | Разрешены только значения: `PENDING`, `PROCESSING`, `SENT`, `FAILED` |
| `chk_notification_outbox_attempts` | `attempts`             | Значение должно быть больше либо равно `0`                           |
